# Installation

    git clone http://github.com/zjrosen1/vim.git ~/.vim
    cd ~/.vim
    ./install.sh

## Need Moar Plugins
###Vundle like its 2015

Add a new plugin to your vimrc

		Plugin 'moar/plugin'
		:PluginInstall

Search for all or specific plugins inside of vim

		:PluginList

		:PluginList ack
